O projeto da Telecom X utilizou inteligência artificial para prever o cancelamento de clientes, tratando dados reais através de uma limpeza rigorosa que removeu valores nulos e colunas sem poder preditivo. Para viabilizar a modelagem, as variáveis de texto foram convertidas em numéricas e os valores financeiros foram padronizados, garantindo que o modelo não fosse distorcido por escalas diferentes. Devido ao baixo número de cancelamentos na base original, aplicou-se a técnica SMOTE para equilibrar os dados sinteticamente, permitindo que os algoritmos aprendessem a identificar padrões de evasão com maior precisão.

A comparação entre modelos demonstrou que o Random Forest superou a Regressão Logística ao capturar comportamentos complexos dos consumidores. A análise final revelou que o tipo de contrato mensal e o curto tempo de relacionamento com a empresa são os maiores gatilhos de saída. Com isso, a solução estratégica propõe o uso do modelo como um sistema de alerta para ações preventivas, como a oferta de incentivos para migração de planos e suporte intensificado para novos clientes, transformando a análise técnica em uma ferramenta direta de aumento de faturamento e retenção
